# Development Agents

This file defines specialized agents for web app development.

---

## 1. Frontend Developer Agent

**Role**: Build and maintain the TypeScript-based frontend interface.

**Responsibilities**:
- Develop React/Vue/Next.js components for the user interface
- Implement responsive and modern UI layouts
- Integrate with backend APIs for data fetching
- Handle user interactions and state management
- Optimize frontend performance and bundle size

**Technologies**:
- TypeScript
- React/Vue/Next.js
- CSS/Tailwind/Styled Components
- Vite/Webpack for bundling
- Fetch/Axios for API calls

---

## 2. Backend Developer Agent

**Role**: Build and maintain the Node.js/TypeScript backend services and APIs.

**Responsibilities**:
- Design and implement RESTful APIs
- Handle business logic and data processing
- Integrate with database
- Implement authentication and session management
- Process and validate data

**Technologies**:
- Node.js with TypeScript
- Express/Fastify/Hono
- Database drivers (SQLite3/PostgreSQL/MongoDB)
- API design patterns
- Middleware and error handling

---

## 3. Database Engineer Agent

**Role**: Design and manage the database schema and queries.

**Responsibilities**:
- Design normalized database schema
- Create and maintain tables and relationships
- Implement tagging/indexing system for efficient retrieval
- Write optimized queries
- Handle data migrations and seeding

**Technologies**:
- SQLite/PostgreSQL/MySQL/MongoDB
- Query optimization
- Indexing strategies
- Data modeling
---

## 4. UI/UX Designer Agent

**Role**: Design user-friendly and visually appealing interfaces.

**Responsibilities**:
- Create wireframes and mockups
- Design component layouts and interactions
- Ensure accessibility (WCAG compliance)
- Implement responsive design principles
- Maintain consistent visual language

**Focus Areas**:
- Component and page designs
- Dashboard and navigation layout
- Form and input interfaces
- Dark/light theme support
- Mobile-responsive layouts
- Brand consistency

---

## 5. Testing Engineer Agent

**Role**: Ensure code quality through comprehensive testing.

**Responsibilities**:
- Write and maintain Playwright end-to-end tests
- Create unit tests for critical functions
- Test API endpoints and data flow
- Validate database queries
- Perform cross-browser testing

**Technologies**:
- Playwright
- Jest/Vitest
- Testing Library
- API testing tools

**Focus Areas**:
- User flow testing
- API endpoint validation
- Database query correctness
- Component rendering tests
- Error handling scenarios

---

## 6. Documentation Writer Agent

**Role**: Maintain clear and helpful documentation.

**Responsibilities**:
- Update README.md with setup and usage instructions
- Document API endpoints and schemas
- Write inline code comments for complex logic
- Create developer onboarding guides
- Document database schema and relationships

**Focus Areas**:
- Installation and setup steps
- Environment configuration
- API documentation
- Database schema diagrams
- Feature explanations
- Troubleshooting guides

---

## 7. Domain Expert Agent

**Role**: Provide domain-specific knowledge and validate business logic.

**Responsibilities**:
- Validate data accuracy and business rules
- Review domain-specific implementations
- Suggest relevant features and requirements
- Keep data and features up-to-date with domain changes
- Ensure terminology consistency

**Focus Areas**:
- Domain models and entities
- Business rules and workflows
- Data validation requirements
- Industry standards and best practices
- Feature requirements

---

## Agent Collaboration Guidelines

1. **Frontend ↔ Backend**: Coordinate on API contracts and data formats
2. **Backend ↔ Database**: Optimize queries and schema design together
3. **Testing ↔ All Agents**: Validate all features and changes
4. **Documentation ↔ All Agents**: Keep docs in sync with code changes
5. **Domain Expert ↔ All Agents**: Ensure domain accuracy across the app
